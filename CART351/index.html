Here is a link to the GitHub repository of the final project. GitHub The project is optimized for synchronizing clients on a local network, and could not be configured to run on a distant Node.js host in time because the synchronization code would need a complete rewrite to account for higher latencies. Therefore it is not possible to experiment with the project without downloading the server code and running it locally. Launch the server by running app.js with node (or by executing launch.command on a mac), and connect with Google Chrome using port 8080.
