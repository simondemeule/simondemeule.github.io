Curvature

Curvature is an experimental hybrid raytracing / raymarching 3D engine written from scratch in C++ with one goal in mind: allowing light to be curved efficiently.

Curvature implements common ray-tracing optomization such as BVH acceleration and multithreading, while also implementing a novel technique for curved light-path integration, which uses distance functions to define safe, spherical zones where some calculations can be skipped in the iterative ray-curving process, resulting in a 1.5x-4x speedup in render times

You can look into Curvature's source code and more detailed documentation here on my GitHub.
