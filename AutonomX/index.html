AutonomX

AutonomX is an open source desktop application that allows artists and students to easily experiment with complex life-like algorithmic processes. A simple graphical interface allows exploration and prototyping, while support for the OSC protocol allows these systems to be integrated into digital art pieces and installations.

AutonomX is a work in progress by Concordia University's Xmodal laboratory; I integrated this team as lead backend architect and developper for the first development cycle.

You can explore the AutonomX's source code and download a compiled executable on the project's GitHub page.
